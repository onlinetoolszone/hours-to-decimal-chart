# Hours to Decimal Chart

A quick reference guide for converting hours and minutes into decimal format.
Useful for timesheets, payroll, and billing calculations.

## 🕐 Conversion Chart

| Hours : Minutes | Decimal Hours |
|----------------|---------------|
| 1:00 | 1.00 |
| 1:15 | 1.25 |
| 1:30 | 1.50 |
| 1:45 | 1.75 |
| 2:00 | 2.00 |
| 2:15 | 2.25 |
| 2:30 | 2.50 |
| 2:45 | 2.75 |

## 📐 How to Convert

Divide the **minutes by 60** to get the decimal.

**Example:** 1 hour 30 minutes → 30 ÷ 60 = 0.50 → **1.50**
